# Are You Dead? App

Боль: Люди, живущие одни, могут умереть и никто не узнает об этом. Особенно актуально для пожилых людей, одиноких людей, людей с хроническими заболеваниями.

Отсюда появилась мысль: **Простое приложение для проверки жизни - если человек не отметился в течение 2 дней, отправляется уведомление экстренному контакту.**

## А дальше?

Идея пришла из вирусного китайского приложения Sileme (四了么), которое стало популярным в Китае. Но мы решили сделать свою версию с улучшениями:
- Бесплатно и без регистрации
- Многоязычность (6 языков)
- Простой и понятный интерфейс
- PWA для установки на любое устройство

## К чему пришли

**Ключевые решения:**
- **PWA вместо нативного приложения** - работает на всех устройствах, не нужно публиковать в магазинах
- **Локальное хранение данных** - приватность, никаких серверов
- **Cloudflare Workers для уведомлений** - простой и надежный способ отправки email
- **Многоязычность** - поддержка 6 языков из коробки
- **Простой UI** - одна кнопка "Нет" (No), один вопрос "Умер?" (Are You Dead?)

**Технологии:**
- Чистый HTML/CSS/JavaScript (без фреймворков)
- Cloudflare Pages для хостинга
- Cloudflare Workers для email уведомлений
- Resend API для отправки писем
- Service Worker для PWA функциональности
